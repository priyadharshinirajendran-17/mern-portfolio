# MERN Portfolio Website

A modern and responsive MERN Stack portfolio website showcasing my skills, projects, and experience.

## Features

- Responsive UI
- React + Vite
- Tailwind CSS
- Express.js Backend
- MongoDB Database
- Contact Form
- Gmail Email Notifications (Nodemailer)
- Resume Download
- Project Showcase

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemailer

## Installation

### Clone Repository

```bash
git clone https://github.com/priyadharshinirajendran-17/mern-portfolio.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside `server`.

```env
MONGO_URI=your_mongodb_connection
PORT=5000
CLIENT_URL=http://localhost:5173
EMAIL_USER=your_email
EMAIL_PASS=your_app_password
```
